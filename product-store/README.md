#Product Store App
Product Store App
A modern e-commerce app built with React, showcasing three state management techniques: Context API + useReducer, Redux Toolkit, and React Query.

✨ Features
Browse products with loading & error states

Product details page

Shopping cart (add, remove, adjust quantity, clear) 

Dark / Light mode toggle (saved in localStorage)

Grid / List view toggle

Search products

Filter by category

Sort by price (low–high, high–low)

Pagination

Toast notifications on add to cart

Fully responsive design

🛠️ Tech Stack
React 18

React Router DOM

Redux Toolkit (cart state)

React Query (API data fetching & caching)

Context API + useReducer (theme & layout settings)

Tailwind CSS (styling)

Lucide React (icons)

Vite (build tool)

🚀 Quick Start
# Clone the repository
git clone https://github.com/your-username/product-store-app.git

# Navigate to project folder
cd product-store-app

# Install dependencies
npm install

# Run development server
npm run dev


open http://localhost:5173
